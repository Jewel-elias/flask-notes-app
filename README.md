# Flask Notes App

A lightweight Flask web application that allows users to sign up, log in, and manage personal notes securely.

**Live Demo:** [https://flask-notes-app-4gv9.onrender.com](https://flask-notes-app-4gv9.onrender.com)

## Features

- User authentication (signup, login, logout)
- Password hashing and session management
- CRUD operations for notes
- SQLite database with SQLAlchemy
- Modular Flask structure

## Tech Stack

- Python, Flask
- SQLAlchemy, SQLite
- HTML, CSS, Jinja2

## Installation

1. Clone the repository
   ```bash
   git clone https://github.com/YOUR_GITHUB_USERNAME/flask-notes-app.git
   cd flask-notes-app
   ```
2. Create a virtual environment and install dependencies
   ```bash
   pip install -r requirements.txt
   ```
3. Run the app
   ```bash
   flask run
   ```
4. Open your browser at `http://localhost:5000`
